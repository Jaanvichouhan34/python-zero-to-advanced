# What is Python?

Python is a high-level, interpreted programming language that is easy to learn
and widely used in software development.

## Why Python?
- Simple and readable syntax
- Beginner friendly
- Large standard library
- Used in web development, data science, AI, automation, and more

## Key Features
- Interpreted language
- Dynamically typed
- Object-oriented
- Platform independent

Python focuses on code readability and allows programmers to write
clear and logical code.
